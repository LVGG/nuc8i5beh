## 注意

### bios74未测试，请自行安装测试

BIOS74的修复/功能：

- 修复了在压力测试期间触发看门狗功能的问题。
- 更新了BIOS代码以通过USB闪存驱动器恢复BIOS。
- 更新的机箱类型默认为0x23。
- 更新了BIOS代码以修复安全问题。
